## 분석 배경

신규 슈팅 게임 프로젝트의 UI/UX 설계를 위해 유사 장르인 FPS 게임 3종(콘코드, 배틀필드 2042, 팰러딘즈)의 메인 로비와 게임 UI를 분석했습니다. 각 게임이 어떤 정보를 어떻게 배치했는지, 그리고 그것이 유저 경험에 어떤 영향을 미치는지 파악하고자 했습니다.
